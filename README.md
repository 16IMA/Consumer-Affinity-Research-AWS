# Consumer-Affinity-Research-AWS
Trabajo final para AWS re/Start

Este proyecto tiene como objetivo analizar la afinidad de los consumidores utilizando servicios de AWS, aplicando técnicas de procesamiento de datos y análisis para identificar patrones de comportamiento.

## 🎯 Objetivos
Analizar datos de consumidores para identificar afinidades y preferencias.
Implementar una arquitectura basada en AWS.
Procesar y visualizar datos de manera eficiente.
🏗️ Arquitectura

## Descripción general de los servicios utilizados en AWS (por ejemplo):

Amazon S3 para almacenamiento de datos
AWS Lambda para procesamiento
Amazon Athena / Redshift para consultas
Amazon QuickSight para visualización

## ⚙️ Tecnologías utilizadas
AWS (S3, Lambda, Athena, etc.)
Python
SQL
Power BI / QuickSight (si aplica)


## 📂 Estructura del proyecto

Consumer-Affinity-Research-AWS/
│── data/              # Datos de entrada
│── scripts/           # Scripts de procesamiento
│── notebooks/         # Análisis exploratorio
│── results/           # Resultados y visualizaciones
│── README.md

## 🚀 Cómo ejecutar el proyecto
Subir los datos a S3.
Ejecutar los scripts de procesamiento.
Consultar los datos usando Athena o Redshift.
Visualizar resultados en QuickSight o herramienta similar.
